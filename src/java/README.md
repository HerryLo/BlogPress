# Java 技术栈

本栏目记录了笔者在 Java 后端开发领域的学习与实践历程，主要围绕 **Java 基础** 与 **Spring Boot 框架** 两大方向展开。

## Java 基础

Java 作为一门成熟稳定的面向对象编程语言，是后端开发的重要基石。这部分内容涵盖了 Java 核心概念、集合框架、多线程编程、JVM 原理等基础知识，帮助夯实编程功底。

## Spring Boot

Spring Boot 是当前 Java 后端开发中最流行的框架之一，它简化了 Spring 应用的初始搭建和配置过程，提供了自动配置、内嵌服务器、健康检查等开箱即用的功能。通过 Spring Boot，可以快速构建高效的 RESTful API 和微服务。
