# Java工程实践文章重构设计

## 概述

对 `Java数据库与JDBC.md` 和 `JavaWeb与Servlet.md` 两篇文章进行结构重组和内容精简，使其更符合学习路径、便于阅读。

## JDBC.md 重构方案

### 目标结构

| 章节 | 内容 | 说明 |
|------|------|------|
| MySQL基础回顾 | 数据库操作、表操作、CRUD | 为后续数据库操作铺底 |
| JDBC使用步骤 | 驱动加载→获取连接→执行SQL→处理结果→关闭资源 | 核心API学习 |
| CRUD完整示例 | 精简代码示例（≤50行） | 实战练习 |
| ORM思想 | 详细说明ORM映射、为什么需要ORM、JDBC的局限性 | 理解数据持久化思想 |

### 内容调整

- **移除章节**：数据库连接池、PreparedStatement防SQL注入、事务管理
- **扩充ORM思想**：详细说明对象与表的映射、常见ORM框架对比
- **缩减CRUD示例**：每个方法代码≤50行
- **增加衔接说明**：每章节说明"为什么有这章、它做什么、对后续的帮助"

### 约束

- 每段代码示例≤50行
- 流程图≤1个
- 每章节描述≥80字

---

## Servlet.md 重构方案

### 目标结构

| 章节 | 内容 | 说明 |
|------|------|------|
| 什么是Servlet | Servlet概念、JavaWeb框架底层 | 为什么需要Servlet |
| 请求流程 | 浏览器→Tomcat→Servlet→响应 | 完整请求链路 |
| Servlet生命周期 | init/service/destroy | 从加载到销毁 |
| 会话管理 | Cookie和Session | 解决HTTP无状态问题，为登录功能铺垫 |
| 注解开发Servlet | @WebServlet简化配置 | 替代web.xml |
| Filter过滤器 | 请求拦截和统一处理 | 编码、权限等 |
| 总结 | 串联全篇，引出Spring MVC | 承上启下 |

### 内容调整

- **移除章节**：HttpServletRequest请求对象、HttpServletResponse响应对象
- **缩减所有代码示例**：每段代码≤50行
- **增加衔接说明**：每章节说明"为什么有这章、它做什么、对后续的帮助"

### 约束

- 每段代码示例≤50行
- 流程图≤1个
- 每章节描述≥80字
