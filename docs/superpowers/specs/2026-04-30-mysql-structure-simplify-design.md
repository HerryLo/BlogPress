# MySQL文章结构精简设计

## 概述

将 MySQL.md 从8个章节精简为5个章节，使结构更清晰、便于阅读。

## 当前结构（8章）

1. 为什么学习MySQL
2. MySQL简介与环境
3. 数据库与表操作
4. 增删改查
5. 常用函数与条件查询
6. 索引
7. 关联查询
8. 事务与隔离级别

## 目标结构（5章）

| 章节 | 内容 | 说明 |
|------|------|------|
| MySQL基础 | 简介、环境、数据库与表操作 | 合并"为什么学习"+"简介与环境"+"数据库与表操作" |
| 增删改查 | INSERT/SELECT/UPDATE/DELETE | 核心SQL操作 |
| 查询增强 | 条件查询、函数、排序分页 | 合并"常用函数与条件查询" |
| 索引与关联 | 索引、JOIN、子查询 | 合并"索引"+"关联查询" |
| 事务 | 事务控制与隔离级别 | 独立章节 |

## 合并逻辑

```
原8章 → 新5章：
- 为什么学习 + 简介与环境 + 数据库与表操作 → MySQL基础
- 增删改查（保留）
- 常用函数与条件查询 → 查询增强（合并到增删改查后）
- 索引 + 关联查询 → 索引与关联
- 事务与隔离级别 → 事务（保留）
```

## 约束

- 每段代码≤50行
- 流程图≤1个
- 每章节有衔接说明
- 结尾引出JDBC
