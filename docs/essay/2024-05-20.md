# 工作几年的一点想法

时代发展面前，个人的力量太弱小，每个人都被时代裹挟，我只是其中的一员而已。

工作已经有8年左右，看着市场行情从火热到低迷，再到现在的惨淡，不知道后面会不会更差，更低。经常翻看网上的文章或消息，知乎、B站、短视频平台各种平台，各种悲观的消息扑面而来，没有最多只有更多。也可能是推送的缘故，我看的都是负面消息，然后它继续为我推送，只会越来越多。

从身边经济学来看，确实市场悲观居多。比如：某某211毕业前同事，上海找工作四月最高20k，最后入职；某某前同事，由于市场行情不好，改行开店；某某前同事，由于市场行情不好改行；等等这样的消息，不胜枚举，只要稍微打听，全是类似的消息。

这样的行情不知道会持续多久，各行各业都在熬，等待外部环境好转，等待市场行情恢复，等待居民负债率降低。不是说未来一定会更好，只是人需要希望，让生活能继续下去，毕竟相比前途昏暗，我还是更喜欢柳暗花明。

不过个人的意志无法左右大环境，客观现实不会因为主观想法而变化，而是“客观现实会因为客观现实变化而变化”，这句话有点像废话，不过确实蛮有道理。

“客观现实”是我们的现实生活，人每天得吃饭，吃饭需要花钱，猪肉多少钱，蔬菜多少钱，每天我们都有消耗和支出，当然也有收入、工资，每个人的生活，工作，交易，共同构成了这个现实的人类社会。

现实社会是动态变化的，每个人都有实实在在的需求和欲望，并不是网络上的一串文字，也不是视频中的一端画面，它是实际存在的，甚至是无法忽视的。

在时代的浪潮面前，每个人都是微不足道的小蚂蚁，但是正因为我们了解自己的渺小，才越需要尊重自己，越要正视自己。我们生活在嘈杂的环境中，不可能不被外界所影响，纷乱的网络环境，各种极端或激进的言论，我们每天都在接受各种信息的洗礼，这些信息对我们的影响是好，还是坏呢？没有能给一个确定的回答。我们能做的只是在这个纷乱的环境，正视自己，尊重自己。

改变自己很难，改变世界更难。先易后难，我们先从能做的事情做起。
