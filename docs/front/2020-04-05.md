# 敲黑板！async/await实际应用

async/await在项目开发中使用相对频繁，那么我们就来聊一聊，async/await开发中的常见应用。

## async/await继发运行

好的！我们先来看下面的示例代码：
```javascript
function syncTime(delay, res) {
    return new Promise((resolve, reject)=> {
        setTimeout(()=> {
            console.log(res);
            resolve();
        }, delay)
    })
}

async function main() {
    for(let i=0 ;i<10; i++){
        await syncTime(1000, i)
    }
}
main();
```
