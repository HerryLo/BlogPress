### **解释：**
这里的上下文对象如下：
```javascript
function fn() {console.log('this指向：', this);}

let Obj = {
    fn: fn
}

window.fn();    // 上下文对象调用, 等价于直接调用 fn()
Obj.fn();       // 上下文对象调用
```
